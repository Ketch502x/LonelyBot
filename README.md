**✔ INSTALATION WITH TERMUX:**
````bash
termux-setup-storage 
apt update 
pkg upgrade
pkg install git -y
pkg install nodejs -y 
pkg install ffmpeg -y 
pkg install imagemagick -y 
pkg install yarn
git clone https://github.com/Ketch502x/LonelyBot
cd LonelyBot
yarn install
npm install
npm update
npm start
````
